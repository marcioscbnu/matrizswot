# SWOT Live MVP

Aplicação React + Vite para preencher uma Matriz SWOT ao vivo e visualizar a viabilidade do projeto em tempo real.

## Como instalar

```bash
npm config set registry https://registry.npmjs.org/
npm install
npm run dev
```

## Build para Vercel

```bash
npm run build
```

Configuração no Vercel:

- Framework: Vite
- Build Command: npm run build
- Output Directory: dist

## Observação

Este projeto não usa bibliotecas externas além de React e Vite.
